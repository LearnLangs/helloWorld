to compile your Go app, follow the instructions below

- make sure you are in the folder your go file is in if not type this in the terminal
```bash pws
    cd ./(foldername)
```
- now you have to mod an initialization, type the command below
``` bash pws
    go mod init filename/m
```
- now you can build your app with the command below and you will have a new file with the .exe at the end
``` bash pws
    go build
```
- finally you can run your app
``` bash pws
    ./filename.exe
```
