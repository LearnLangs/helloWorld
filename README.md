# helloWorld

This is a learning project demonstrating "Hello World" implementations in multiple programming languages.

## Languages Covered

- **[C](c/)** - Basic C examples with compilation instructions
- **[Go](go/)** - Simple Go module with run/build steps
- **[JavaScript](js/)** - Node.js script example
- **[Python](python/)** - Python script demonstration

Each directory contains a minimal "Hello World" program along with its own README.md for specific build and run instructions.

## Getting Started

Clone the repository and navigate to any language directory to explore the examples.

```bash
git clone https://github.com/krush-ghod/helloWorld.git
cd helloWorld
```

Since this is a learning project, each stage would have its own repo.
